### 💡 구현 아이디어
N을 최소 개수의 제곱수 합으로 표현으로 표현하기 때문에 이전 결과를 활용한 DP를 적용한다.
1. 주어진 N의 제곱근을 구한다.(sqrt)
2. dp는 작은 값부터 채워간다.
3. sqrt는 1씩 빼가며 반복하고 0보다 큰 동안 실행한다.
4. dp[i]는 `dp[i - 1] + 1`과 `dp[i - sqrt * sqrt] + 1`  중 최소값이 된다.
