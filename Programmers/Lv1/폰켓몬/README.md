# Programmers 폰켓몬
### [문제 바로가기](https://school.programmers.co.kr/learn/courses/30/lessons/1845)
난이도 : Lv1 

### 💡 아이디어
---
입력된 배열에서 폰켓몬의 중복을 없애기 위해 `HashSet`을 적용하였다. 
 <br/>

#### 구현
1. 입력된 배열 길이의 1/2과 폰켓몬의 중복을 없앤 갯수를 비교하여 작은 값을 결과값으로 한다. <br/><br/>


### ✏️ 마치며
---
중복을 없애야 할 경우에는 `HashSet`을 적용한다. <br/>
